# Пешеходный переход

Интерактивный пешеходный светофор с автоматическим переключением и таймером обратного отсчёта.

https://arsengalst.github.io/Pedestrian_crossing/

## Что умеет

- Автоматически переключается между красным и зелёным сигналом
- Показывает обратный отсчёт (30 сек на красном, 20 сек на зелёном)
- Кнопка вызова перехода - зелёный включится после окончания текущей фазы
- Звуковой сигнал при переходе на зелёный (через браузерный Audio API)

## Технологии

HTML, CSS, JavaScript (без библиотек)

## Запуск

Открыть `index.html` в любом браузере.
